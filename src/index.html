Antes de cifrar, recuerda poner un número de desplazamiento

Modo de uso:

Un "número de desplazamiento" es la cantidad de letras que se correrá el avecedario. de esta forma obtienes tu cifrado, basado en el"cifrado Cesar". Así de simple. Puedes cifrar y descifrar letras mayúsuculas, letras minúsculas y números.
Si no has escuchado antes sobre el cifrado Cesar puedes visitar este link de wikipedia.
Puedes tener una misma clave para todas tus cuentas, con números de desplazamiento diferentes para cada una.
