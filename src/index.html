Modo de uso:

Esta página basa su funcionamiento en el "codigo Cesar" Para usarla correctamente primero debes escoger un número de desplazamiento, que representa cuantas letras de desface tendrá el alfabeto, y con ello tu contraseña.

Puedes cifrar y descifrar letras mayúsuculas, letras minúsculas y números.

Si no has escuchado antes sobre el cifrado Cesar puedes visitar este link de wikipedia.

Puedes tener una misma clave para todas tus cuentas, con números de desplazamiento diferentes para cada una.

Antes de cifrar, recuerda poner un número de desplazamiento
